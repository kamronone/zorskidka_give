# Установка
Для корректной работы требуется python версии 3.12

###   .env

    `TOKEN=''
     HOSTNAME_DB=''
     USER_DB=''
     PASSWORD_DB=''
     DATABASE_DB=''`
##### Установка зависимостей:

    `pip install -r requirements.txt`

#### Запуск:

    `python app.py`
